#### CK3 Barony ID modifier.

## How to Build
Install Rust,
```
git clone https://github.com/younghyun1/barony_id_modifier.git
```
go inside the directory on a terminal and then
```
cargo build --release
```

This will create an executable in the directory ./target/release/.

## How to Use
Please refer to
```
bidm --help
```
